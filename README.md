# Weisheit auf Deutsch

> **Немецкий язык через Книгу Притчей Соломоновых (Мишлей)**
> Интерактивный курс · 31 урок · Уровни A1 → B1

---

## Метод Bible Language Immersion

### Главная идея

Язык изучается через повторяющийся мудрый текст, который:

- короткий и ритмичный
- содержит сильные образы
- легко запоминается

**Мишлей идеально подходит для этого, потому что:**

- короткие стихи с чёткими противопоставлениями
- много повторяющихся слов
- практические жизненные ситуации

Это делает книгу идеальной базой для глубокого языкового погружения.

---

## 5 принципов метода

### 1. Спиральное повторение

Каждый урок повторяет лексику, грамматику и темы предыдущих глав.

| Урок | Содержание |
|------|------------|
| Урок 1 | Weisheit, hören |
| Урок 5 | + повтор слов из уроков 1–4 |
| Урок 10 | + повтор всего пройденного |

Это создаёт долговременную память.

---

### 2. От понимания к речи

Каждый урок проходит через 4 стадии:

```
Input          →  Recognition  →  Controlled output  →  Free speech
(чтение стиха)    (узнавание       (простые              (свободные
                   слов)            предложения)           ответы)
```

Подход соответствует современной методике **Communicative Language Teaching**.

---

### 3. Моральный контекст

Каждая грамматическая тема привязывается к теме Мишлей.

**Пример:**

| Грамматика | Пример | Тема |
|------------|--------|------|
| Модальные глаголы | *Du sollst hören. Du kannst lernen.* | Слушать мудрость |

---

### 4. Повторение ключевых слов

В Мишлей есть ядро слов, которые проходят через весь курс:

| Слово | Значение |
|-------|----------|
| die Weisheit | мудрость |
| der Narr | глупец |
| der Weg | путь |
| das Herz | сердце |
| der Rat | совет |
| hören | слушать |
| lernen | учиться |

---

### 5. Жизненные ситуации

Каждый стих превращается в практическую ситуацию.

**Пример:**

> Стих о ленивом человеке → Упражнение:
> *Dein Freund arbeitet nicht. Was sagst du?*

---

## Структура одного урока

```
1. Weisheitsfrage    — вопрос урока
                       «Warum ist Weisheit wichtig?»

2. Kontext           — простое объяснение главы

3. Sprüche           — 3–4 стиха на иврите, русском и немецком

4. Wortschatz        — 10–15 ключевых слов

5. Grammatik         — одна грамматическая тема

6. Übungen           — 4 типа упражнений
                       · лексика
                       · грамматика
                       · работа со стихами
                       · ситуации

7. Dialog            — короткий диалог

8. Weisheitsreflexion — вопрос о смысле стиха
```

---

## Типы упражнений

### 1. Fill the verse
```
Der Weise hört ____.
```

### 2. Opposites
```
Der Weise ≠ der ___.
```

### 3. Moral choice
```
Wer ist klug?
  A) ...
  B) ...
```

### 4. Situation answer
```
Du hörst einen guten Rat. Was machst du?
```

### 5. Build a proverb
```
Слова вразброс: Der / hört / Weise
→ собери правильное предложение
```

---

## Почему этот метод работает

Он соединяет **язык + мудрость + повторение + контекст + рефлексию**.

Это делает обучение:

- **более глубоким** — текст несёт смысл, а не просто слова
- **более запоминающимся** — эмоциональный и образный контекст
- **более мотивирующим** — каждый урок содержит ценную идею

## Программа курса

**31 урок = 31 глава Мишлей**

| Этап | Главы | Уровень |
|------|-------|---------|
| Начало | 1–10 | A1 |
| Развитие | 11–20 | A2 |
| Углубление | 21–31 | B1 |

### Грамматика по главам

| Глава | Тема | Грамматика |
|-------|------|------------|
| 1 | Начало мудрости | Глагол *sein* в настоящем времени |
| 2 | Поиск мудрости | Артикли: *der, die, das* |
| 3 | Доверие к Богу | Модальные глаголы: *können, müssen, sollen* |
| 4 | Ценность мудрости | Глагол *haben* в настоящем времени |
| 5 | Предостережение | Именительный и винительный падежи |
| 6 | Трудолюбие | Отделяемые приставки |
| 7 | Благоразумие | Перфект: прошедшее время |
| 8 | Зов мудрости | Претерит: *sein* и *haben* |
| 9 | Пир мудрости | Прилагательные после артикля |
| 10 | Противоположности | Множественное число существительных |
| 11 | Честность | Предлоги с дательным падежом |
| 12 | Слово и дело | Предлоги с винительным падежом |
| 13 | Мудрость в доме | Двойные предлоги (*Wechselpräpositionen*) |
| 14 | Добрые слова | Возвратные глаголы |
| 15 | Пути Господни | Сравнительная степень прилагательных |
| 16 | Гордость и смирение | Превосходная степень прилагательных |
| 17 | Семейные связи | Придаточные предложения с *dass* |
| 18 | Язык | Относительные придаточные предложения |
| 19 | Бедность и богатство | Страдательный залог (*Passiv*) |
| 20 | Вино и справедливость | Конъюнктив II: вежливость и желания |
| 21 | Сердце царя | Инфинитивные конструкции с *zu* |
| 22 | Доброе имя | Родительный падеж |
| 23 | Умеренность | Словообразование: приставки |
| 24 | Зависть и мудрость | Словообразование: суффиксы |
| 25 | Речи царя | Числительные и даты |
| 26 | Глупость | Прямая и косвенная речь |
| 27 | Хвала и дружба | Условные предложения (*wenn*) |
| 28 | Закон | Причастие I и II |
| 29 | Дисциплина | *Futur I* и *Futur II* |
| 30 | Слова Агура | Союзы: *weil, obwohl, damit* |
| 31 | Добродетельная жена | Обзор: все падежи и времена |
